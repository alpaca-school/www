📋 1. システム概要

「知識・スキル・文化」をトークン化し、QRコードスキャン経由でブロックチェーン上に自動記録するシステム。参加者の出品登録・SNS投稿・推薦などの「参加行為」がリアルタイムでトークン化されます。

項目内容
対象時期2026年8月〜12月
ブロックチェーンPolygon Testnet（2026）→ Solana Mainnet（2027〜）
参加者層出品者・購入者・運営スタッフ（推定100-200名）
処理時間QRスキャン後 3〜5秒でトークン発行
人生科モジュールModule 6 - 価値流通の実装実験

主要な特徴として、ゼロトラスト検証（AI自動判定）、即座性（3〜5秒）、透明性（改ざん不可能な記録）、オフライン対応（事後同期設計）があります。

📱 3. QRコード仕様

各行為に対して一意のQRコードが生成されます。ペイロードにはイベントID、行為種別、nonce（使い捨て）、AI判定要件、コントラクトアドレスが含まれます。

{ "event_id": "furima_2026_okinawa", "action_type": "item_registration|sns_post|...", "nonce": "abc123def456...", "required_ai_checks": [ "identity_verification", "authenticity_check", "isolation_score_adjustment", "risk_detection" ] }
項目仕様
使用回数1QRコード = 1度のみスキャン可
有効期間生成から30日間
配布形態オンライン（アプリ内）/ 紙プリント / ハイブリッド

🔌 4. 主要APIエンドポイント

🧠 5. AI判定エンジン詳細

レイヤー判定内容閾値
Layer 1: 本人確認ウォレット署名（ECDSA）・生体認証・デバイスID・過去行動パターン≥ 0.95
Layer 2: 真正性新規性検査・SNSアカウント実績・推薦元信用・位置情報≥ 0.85
Layer 3: 孤立スコア社会的関与(40%) + コミュニティ貢献(30%) + 信用蓄積(30%)1.0x〜2.0x
Layer 4: リスク検出短時間多数スキャン・同一デバイス大量アカウント・不可能な位置変化< 0.7

孤立スコア計算例:

社会的関与度 = 0.8 × 0.4 = 0.32 コミュニティ貢献 = 0.6 × 0.3 = 0.18 信用蓄積度 = 0.9 × 0.3 = 0.27 ───────────────────── 合計倍率 = 1.0 + 0.77 = 1.77x → 基本10トークン × 1.77 = 17.7トークン

⛓️ 8. スマートコントラクト設計

主要な3つの関数と、ロールベースアクセス制御で構成されます。

// トークン発行 mintTokens(address, amount, actionType, timestamp) → onlyMinter / noReentrancy // 孤立スコア更新 updateIsolationScoreMultiplier(address, newMultiplier) → onlyOracle / 100-200 (1.0x-2.0x) // 二重使用防止 validateActionId(actionId) → usedActionIds mapping
ロール権限例
Ownerロール付与・初期設定法人代表
MintermintTokens() 実行APIサーバー
Oracleスコア更新AI判定エンジン
Pauser緊急停止セキュリティチーム

🔒 9. セキュリティモデル

ゼロトラスト原則に基づき、全ての外部入力を「信頼できない」と仮定し、複数層の検証を通過した後のみ内部状態を変更します。

検証対象検証方法
QRペイロードJSON schema + nonce チェック
ウォレット署名ECDSA 暗号検証
タイムスタンプUTC ±3分ウィンドウ
デバイスID事前登録チェック
二重使用usedActionIds mapping
権限分離onlyMinter / onlyOracle

📡 11. オフライン対応

沖縄のインターネット環境を想定し、オフライン時はローカル（SQLite）に一時記録し、復帰時に自動同期を行います。

【オフライン時】 QRスキャン → ローカル一時記録（SQLite） → UI: 「同期保留中」表示 【復帰時】 ① ローカル記録をAPI送信 ② 重複チェック（処理済みならスキップ） ③ ブロックチェーンに記録 ④ 「✅ 同期完了」通知

QRスキャンが不可能な場合は「手動報告フォーム」から入力し、オペレーターがAI判定を手動トリガーします。
