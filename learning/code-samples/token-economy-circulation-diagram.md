## 詳細説明

### 第1層：ユーザー行動と基本報酬

ユーザーの具体的な行動がトークン獲得の源泉です。各行動タイプには固定の基本報酬が設定されています：

- **出品登録** (item_registration): 20 トークン
  - 沖縄文芸フリマへの商品出品、店舗登録など
  - 最も価値の高い行動（経済活動の直接起動）

- **SNS投稿** (sns_post): 15 トークン
  - ハッシュタグ付きSNS投稿で沖縄文芸フリマを宣伝
  - 社会的関与と認知度向上に貢献

- **推薦** (recommendation): 10 トークン
  - 友人・知人をプラットフォームに招待
  - ネットワーク拡大のインセンティブ

- **参加** (attendance): 5 トークン
  - イベント参加、オンライン学習完了
  - コミュニティ参加の基礎報酬

- **サポート** (support): 8 トークン
  - 他ユーザーのQ&A回答、コンテンツ評価
  - コミュニティ相互扶助に対する報酬

### 第2層：孤立スコア乗数メカニズム

**最もユニークな特徴**：社会的つながりの強さに応じて報酬を1.0倍～2.0倍に増幅します。

孤立スコア乗数の計算式：
```
孤立スコア乗数 = 1.0 + (社会的関与×40% + コミュニティ参加×30% + 信用蓄積×30%)
範囲: 1.0x ～ 2.0x
```

**社会的関与 (40%)**
- 友人紹介数：何名がユーザーから紹介されたか
- フォロー数：プラットフォーム内でのネットワークサイズ
- コラボレーション実績：他ユーザーとの共同プロジェクト

**コミュニティ参加 (30%)**
- イベント参加頻度：週1回～月数回の参加状況
- 投票参加率：DAO投票、コミュニティ意思決定への参加
- ギルド加入：複数のサブコミュニティ加入状況
- 寄付実績：他ユーザーやプロジェクトへの支援

**信用蓄積 (30%)**
- 完了率：出品～売買成立までのプロセス完了度
- 評価スコア：他ユーザーからの評価の平均値
- 報告期限厳守：提出期限を守る実績
- 返金実績：返金・キャンセルが全くない実績

この設計により：
- **初心者も参加可能**：最低 1.0x の倍率が保証される
- **参加を続けるほど報酬増加**：3ヶ月で 1.3x、6ヶ月で 1.6x、12ヶ月で 2.0x に到達可能
- **社会的孤立を緩和**：つながりが少ない人ほど倍率上昇の余地が大きい

### 第3層：月次トークン供給源

Phase 1 では毎月 ¥10M (10,000,000トークン) が新規発行されます。

| 供給源 | 金額 | 割合 | 用途 |
|--------|------|------|------|
| ユーザー報酬 | ¥6.0M | 60% | 個人行動への直接報酬 |
| コミュニティ奨励 | ¥2.0M | 20% | ギルド・グループへの集合報酬 |
| インフラ費 | ¥1.2M | 12% | API・コントラクト運用 |
| リザーブ | ¥0.8M | 8% | 市場安定化・緊急対応 |

### 第4層：ユーザーフロー別の獲得パターン

#### スターティングユーザー（初月～3ヶ月）
- 月獲得: 50～100トークン
- 3ヶ月累積: 150～300トークン (¥15,000～¥30,000相当)

### 第5層：トークン換金パス

トークン獲得後、3つの異なるパスで価値を実現できます：

**パスA: ローカル流通** 
- カフェ、農産物市場、工芸品などの沖縄内商品購入
- 地域経済活性化に直結
- 年¥165,000相当の消費拡大を実現

**パスB: 投資機会**
- 沖縄文芸フリマの IPO プリセール参加
- 株式化後の配当・成長期待（2030年予定）
- 3年後の企業評価¥5M以上への投資リターン期待

**パスC: 社会貢献**
- 教育基金、福祉施設、環境保全への寄付
- 2030年以降 DAO 投票権の確得

### 第6層：マクロ経済循環

3つの Phase で段階的にスケーリング：

```
Phase 1 (2026-2028)  → 月¥10M、参加者 1,000名、平均孤立スコア 1.3x

```

Phase 3 では、年間 ¥6,000M = **沖縄県 GDP の約 5%** に相当する新規価値循環が創出されます。

## システムの自律性

このトークン経済循環システムの最大の特徴は、**外部による中央管理を最小化**した自律的な価値循環にあります：

1. **分散的な価値判断**
   - 各ユーザーが独立して行動を選択
   - AI判定で透明な基準による報酬付与

2. **ネットワーク効果による自己強化**
   - 参加者が増えるほど孤立スコア上昇の機会が増加
   - 社会的つながりが強いほど報酬が増加
   - 参加インセンティブが自動的に高まる

3. **多様な価値実現**
   - 消費、投資、社会貢献など多数のパス
   - ユーザーが自由に選択可能
   - 中央による強制がない

4. **スケーラビリティ**
   - 参加者数と報酬は比例関係（線形）
   - インフラ費も参加規模に応じて自動調整可能
   - Phase 1→2→3 への遷移が自然

このシステムは、**Licklider の「銀河間コンピュータネットワーク」ビジョン**を経済レイヤーで実装したものです。沖縄全体の価値循環ネットワークが、中央集約的な管理なしに自律的に成長していく構造を目指しています。
